# Customer Segmentation (Retail)

This repository contains a set of Jupyter notebooks that walk through customer segmentation analysis using the **Online Retail** dataset (Kaggle). The goal is to perform data preprocessing, exploratory data analysis (EDA), and customer segmentation using **RFM analysis** and **K-Means clustering**.

---

## 📁 Repository Structure

- `data/` - Dataset files and documentation:
  - `raw/` - Raw dataset files (original source data)
  - `processed/` - Cleaned/processed CSV files generated by the preprocessing notebook
  - `cluster/` - Final customer segmentation output (cluster assignments)
  - `dataset.md` - Detailed documentation of the datasets
- `doc/` - Documentation and presentation files:
  - `final_insight.md` - Final insights and conclusions from the analysis
  - `marketing_campign.md` - Marketing campaign recommendations based on clusters
  - `Customer Segmentation Using RFM & K-mean clustering.pdf` - Slide presentation of the project
- `notebooks/` - Jupyter notebooks for each analysis step:
  - `1_data_preprocessing.ipynb` - Clean raw data and create a consistent processed dataset
  - `2_data_exploration.ipynb` - Explore trends, time series, and customer behavior
  - `3_RFM_and_Kmean_clustering.ipynb` - Perform RFM scoring and cluster customers using K-Means
  - `4_EDA_for_each_cluster.ipynb` - Analyze each cluster for behavior and insights
- `Requirements.txt` - Python dependencies required to run the notebooks

---

## 📖 Documentation

- `doc/final_insight.md` - Summary of key findings and insights from the customer segmentation analysis
- `doc/marketing_campign.md` - Targeted marketing campaign strategies for each customer segment
- `doc/Customer Segmentation Using RFM & K-mean clustering.pdf` - Comprehensive slide presentation covering the entire project methodology, results, and recommendations

## ✅ Getting Started

### 1) Install dependencies

From the repository root:

```bash
python -m venv .venv
# Activate the virtual environment:
# Windows (PowerShell): .\.venv\Scripts\Activate.ps1
# Windows (cmd): .\.venv\Scripts\activate.bat
# macOS / Linux: source .venv/bin/activate

pip install -r Requirements.txt
```

### 2) Launch Jupyter

```bash
jupyter lab
```

Then open and run the notebooks in order (1 → 2 → 3 → 4).

---

## 🧠 Recommended Notebook Execution Order
1. **`1_data_preprocessing.ipynb`** – Clean raw data and save processed output (CSV).
2. **`2_data_exploration.ipynb`** – Explore trends, visualize revenue/time patterns, and gain insights.
3. **`3_RFM_and_Kmean_clustering.ipynb`** – Build RFM scores and perform K-Means customer segmentation.
4. **`4_EDA_for_each_cluster.ipynb`** – Analyze cluster behavior and generate recommendations.

---

## 📌 Notes

- The notebooks assume the dataset is available at `data/raw/Online_Retail.xlsx` (Kaggle dataset).
- Processed outputs are written to `data/processed/` and `data/cluster/`.
- If you replace the raw data, rerun `1_data_preprocessing.ipynb` first to regenerate processed datasets.

---

## 🧩 Dataset Source

The dataset is based on the Kaggle "Online Retail" dataset:
